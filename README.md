## Overview

itamarimor2009 Esports Dashboard brings together the key data esports fans usually have to chase across several sites. It aggregates live match information, current standings, and broadcast access for three major competitive games: Fortnite, Valorant, and CS2. The result is a single, straightforward place to follow active tournaments without constantly switching tabs or services.

The design emphasizes fast loading and easy reading, so you can check ongoing matches, review leaderboard movement, and look back at previous champions with minimal effort. Live updates and a validated historical archive sit side by side, giving the dashboard both immediate usefulness and long-term context.

## How to Use

1. Launch the dashboard in your browser.
2. Use the top navigation to switch between Fortnite, Valorant, and CS2.
3. Follow live matches, including score updates and current status.
4. Select the broadcast icon next to a match to open the official stream.
5. Move to the leaderboard section to inspect standings.
6. Explore the champions archive to review winners and event outcomes.

## Settings

All configurable options live in the root-level `config.json` file. There you can tune match refresh timing, choose the default game shown on load, and set broadcast link behavior. No database setup and no external API credentials are required.

## FAQ

**How frequently is match data refreshed?**  
By default, match status and leaderboard information update every 30 seconds. That interval can be changed in the configuration file.

**Can the dashboard support more games?**  
Right now it is centered on Fortnite, Valorant, and CS2. Additional titles may be added later depending on community demand.

**Is there a native mobile app?**  
No separate mobile app is available. The web UI is responsive and is intended to work well in mobile browsers.

**How do I report a problem?**  
Open an issue in the GitHub repository and include the issue details along with your browser version.
